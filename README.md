## What This Is
Complete DevOps transformation of an e-commerce platform including:
- Docker containerization
- CI/CD with Jenkins & GitHub Actions
- AWS infrastructure with Terraform
- Monitoring with Prometheus/Grafana/ELK
- Automated backups and scripts

## Tech Stack
- **Frontend**: React
- **Backend**: Node.js + Express
- **Database**: PostgreSQL (AWS RDS)
- **Cache**: Redis (AWS ElastiCache)
- **Cloud**: AWS
- **IaC**: Terraform
- **Config**: Ansible
- **Monitoring**: Prometheus, Grafana, ELK, Nagios
